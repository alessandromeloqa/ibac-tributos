# Envelope de Contribuição - Subdomínio

## Instalação

### 1. Criar Subdomínio
- Acesse o painel da sua hospedagem
- Crie um subdomínio (tributos-lorena.ibacvsj.com.br/)
- Aponte para a pasta `public_html`

### 2. Upload dos Arquivos
- Faça upload de todos os arquivos desta pasta para o diretório do subdomínio
- Estrutura final:
  ```
  tributos-lorena.ibacvsj.com.br/
  ├── index.html
  ├── .htaccess
  ├── robots.txt
  └── README.md
  ```

### 3. Configurações Opcionais

#### SSL/HTTPS
- Ative o certificado SSL no painel da hospedagem
- Descomente as linhas do HTTPS no arquivo `.htaccess` se necessário

#### Personalização
- Edite o arquivo `index.html` para alterar:
  - Nome da igreja (linha 32)
  - Logo da igreja (linha 31)
  - Chave PIX (linha 120)
